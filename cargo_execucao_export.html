⚙️ Automação — Como a tabela é populada

Power Automate (Flow): Roda diariamente ou sob demanda.
1. Consulta saes_contratoclts (ativos) → gera 1 registro/mês por contrato
2. Consulta saes_contratopfs (ativos) → gera 1 registro/mês
3. Consulta saes_contratopjs (pagamentos) → 1 registro por pagamento
4. Popula saes_execucao com: projeto, PEP, ID registro, depto, nome, CPF, valor, datas, competência, status

Alternativa JS: O dashboard calcula em tempo real a partir dos contratos (sem tabela extra), mas com tabela própria fica mais rápido para consultas e exportações.
