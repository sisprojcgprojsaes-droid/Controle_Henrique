🔧 Nota Técnica — Implementação

Storage: Dataverse annotation (Notes/Annotations)

Campos usados:

CampoUso
objectidID do contrato (saes_contratopfid / saes_contratocltid)
subjectTipo do anexo (ex: "contrato_assinado", "diploma", "parecer")
notetextDescrição livre do usuário
filenameNome do arquivo original
documentbodyConteúdo em Base64
mimetypeMIME type (application/pdf, image/jpeg, etc.)
filesizeTamanho em bytes

API:

// Upload
POST /api/data/v9.2/annotations
{
  "[email protected]": "saes_contratopfs({id})",
  "subject": "contrato_assinado",
  "notetext": "Contrato assinado em 15/04/2024",
  "filename": "Contrato_CLT-2024-0045.pdf",
  "documentbody": "<base64>",
  "mimetype": "application/pdf"
}

// Listar
GET /api/data/v9.2/annotations
  ?$filter=_objectid_value eq '{contrato_id}'
  &$select=subject,notetext,filename,filesize,mimetype,createdon
  &$orderby=createdon desc

Limite: 5MB por annotation (padrão Dataverse). Para arquivos maiores, considerar SharePoint integration futuramente.
